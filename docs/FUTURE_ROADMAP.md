# 方言宝未来扩展路线图 (Future Roadmap)

> 📅 创建日期：2026-01-28
> 📝 版本：v1.0
> 🎯 目标：Dialect Master 长期发展规划，从社区迈向智能文化平台。

随着社区基础设施的完善 (Phase 1-6 完成)，Dialect Master 将向着更智能、更实时、更多元的方向发展。以下是下一阶段的重点规划：

## Phase 7: 实时互动与多媒体增强 (Real-time & Multimedia)
*目标：打破异步交流的限制，提供沉浸式的方言交流体验。*

- [ ] **方言直播间 / 语音聊天室**
    - 利用 WebRTC 技术实现多人实时语音聊天。
    - 开设主题房间，如“粤语角”、“评书会”、“方言KTV”。
- [ ] **短视频功能**
    - 支持上传 15-60s 的方言短视频，捕捉生活中的方言瞬间。
    - 引入视频流播放体验。
- [ ] **富文本编辑器升级**
    - 发帖支持插入图片画廊、投票组件、表情包。

## Phase 8: AI 深度集成与辅助学习 (AI Integration)
*目标：利用 SenseVoice 及更多 AI 模型，从“社区”进化为“智能学习平台”。*

- [ ] **AI 方言纠音教练**
    - 用户录音 -> AI 分析发音准确度、语调 -> 给出评分和改进建议。
- [ ] **方言互译与字幕生成**
    - 自动为方言视频/音频生成标准普通话字幕。
    - 不同方言之间的翻译辅助。
- [ ] **智能内容推荐**
    - 基于用户兴趣图谱和行为数据，精准推荐感兴趣的方言内容。

## Phase 9: 方言知识图谱与文创 (Knowledge & Culture)
*目标：沉淀文化资产，建立权威的方言数据库。*

- [ ] **共建方言词典 (Wiki)**
    - 用户协作编辑方言词条、例句、发音。
- [ ] **可视化方言地图**
    - 基于 GIS 技术，在地图上点亮用户的家乡方言。
    - 点击地图区域即可收听当地的方言录音。
- [ ] **方言文化遗产档案**
    - 针对濒危方言的抢救性录音存档项目。

## Phase 10: 多端生态 (Ecosystem)

- [ ] **移动端 App (React Native / Flutter)**
    - 开发 iOS 和 Android 原生应用。
- [ ] **微信小程序**
    - 利用社交裂变优势进行快速传播。
